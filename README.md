# BalleCoffe

Site desenvolvido focado em layout, responsividade e pequenas interações para tornar a navegação mais fluida e agradável.
O projeto simula a página de uma cafeteria, explorando identidade visual e estrutura de uma site simple.

## Preview
Acesse os site: https://livia2710.github.io/BalleCoffe/

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Conceitos Aplicados
- Layout responsivo
- Estrutura semântica de HTML
- Estilização com CSS
- Interações simples com JavaScript
